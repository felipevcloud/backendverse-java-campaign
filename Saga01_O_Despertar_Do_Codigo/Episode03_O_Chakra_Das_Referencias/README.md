# Episode 03 — O Chakra das Referências: Ecos do Heap

## Objetivo
Praticar a diferença entre valores primitivos, referências, objetos, arrays e null em Java.

## Conceitos praticados
- tipos primitivos
- variáveis de referência
- objetos em memória
- arrays de primitivos
- arrays de objetos
- null
- if / else
- loops

## Descrição da missão
O programa simula um laboratório da Academia Backend onde o aprendiz observa como valores primitivos são copiados, como referências podem apontar para o mesmo objeto e como arrays de objetos podem conter posições vazias.

## Como executar
1. Abrir o projeto no IntelliJ.
2. Executar a classe `LaboratorioHeap`.
3. Verificar a saída no console.

## Resultado esperado
O console deve mostrar diferenças entre cópia de valores primitivos, referências para objetos, arrays preenchidos e posições null tratadas com segurança.
