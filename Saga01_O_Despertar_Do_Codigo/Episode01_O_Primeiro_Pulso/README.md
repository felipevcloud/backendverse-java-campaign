# Episode 01 — O Primeiro Pulso

## Objetivo
Criar meu primeiro programa da Backendverse Java Campaign, praticando a estrutura básica de um programa Java.

## Conceitos praticados
- class
- main
- System.out.println
- variáveis simples
- if / else
- loops

## Descrição da missão
O programa simula a entrada de um aprendiz na Academia Backend. Ele exibe mensagens no console, executa pulsos de inicialização com loop e verifica se o aprendiz está pronto para iniciar a campanha.

## Como executar
1. Abrir o projeto no IntelliJ.
2. Executar a classe `PrimeiroPulso`.
3. Verificar a saída no console.

## Resultado esperado
O console deve mostrar o nome do aprendiz, o nível inicial, três pulsos de inicialização e uma mensagem final autorizando o início da jornada.
