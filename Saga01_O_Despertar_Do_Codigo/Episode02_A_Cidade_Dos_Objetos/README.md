# Episode 02 — A Cidade dos Objetos

## Objetivo
Praticar os conceitos iniciais de classes e objetos em Java.

## Conceitos praticados
- classes
- objetos
- instanciação com new
- atributos de instância
- métodos simples
- operador ponto
- separação entre modelo e execução

## Descrição da missão
O programa registra aprendizes da Academia Backend. Cada aprendiz é criado como um objeto da classe `AprendizBackend`, possui seus próprios atributos e executa métodos para se apresentar e mostrar seu status.

## Como executar
1. Abrir o projeto no IntelliJ.
2. Executar a classe `AcademiaBackend`.
3. Verificar a saída no console.

## Resultado esperado
O console deve mostrar dois aprendizes diferentes, cada um com nome, especialidade e nível próprios.
