# Episode 05 — O Arsenal do Sensei: A Provação dos Cem Alvos

## Objetivo
Construir um programa Java mais completo, integrando métodos, parâmetros, retornos, arrays, loops, validações e testes manuais.

## Conceitos praticados
- métodos com parâmetros
- métodos com retorno
- atributos privados
- arrays de objetos
- arrays de valores primitivos
- for
- while
- if / else
- contadores
- testes manuais
- organização do fluxo de execução

## Descrição da missão
O programa simula uma provação da Academia Backend. Três alvos com resistências diferentes recebem ataques até serem destruídos. Antes da provação principal, os comportamentos da classe `AlvoTreinamento` são verificados por testes manuais.

## Estrutura
- `AlvoTreinamento`: representa e controla o estado de um alvo.
- `TesteAlvoTreinamento`: testa os métodos da classe modelo.
- `ProvacaoDoArsenal`: executa a missão principal.

## Como executar

### Testes
Execute a classe `TesteAlvoTreinamento`.

### Programa principal
Execute a classe `ProvacaoDoArsenal`.

## Resultado esperado
Os testes devem exibir resultados PASS ou FAIL. A provação principal deve destruir os três alvos e apresentar um relatório com a quantidade de ataques realizados.
