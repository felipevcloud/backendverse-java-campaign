# Episode 06 — A Biblioteca dos Sábios: O Grimório Infinito

## Objetivo
Aprender a utilizar classes prontas da biblioteca Java, especialmente o ArrayList.

## Conceitos praticados
- import
- java.util
- ArrayList
- add
- get
- size
- remove
- isEmpty
- busca em coleções
- validação de índices
- consulta à API Java

## Descrição da missão
O programa representa um grimório de técnicas da Academia Backend. Técnicas podem ser adicionadas, listadas, buscadas e removidas durante a execução.

## Estrutura
- `TecnicaRegistrada`: representa uma técnica.
- `BibliotecaTecnicas`: administra o ArrayList.
- `ArquivoDosSabios`: executa o fluxo principal.

## Métodos da API utilizados
- `add`: adiciona um elemento ao ArrayList.
- `get`: retorna o elemento armazenado em uma posição.
- `size`: retorna a quantidade de elementos.
- `remove`: remove um elemento pela posição.
- `isEmpty`: informa se o ArrayList está vazio.

## Como executar
1. Abrir o projeto no IntelliJ.
2. Executar a classe `ArquivoDosSabios`.
3. Verificar as operações exibidas no console.

## Resultado esperado
O programa deve adicionar, listar, buscar e remover técnicas, além de tratar uma posição inválida sem interromper a execução.
