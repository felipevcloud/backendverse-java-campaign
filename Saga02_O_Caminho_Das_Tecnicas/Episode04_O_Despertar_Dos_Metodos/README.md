# Episode 04 — O Despertar dos Métodos: Técnicas que Movem Objetos

## Objetivo
Praticar métodos de instância, parâmetros, retorno, atributos privados e alteração de estado interno dos objetos.

## Conceitos praticados
- métodos de instância
- métodos com parâmetros
- métodos com retorno
- atributos privados
- validação com if / else
- alteração de estado interno
- objetos com comportamento

## Descrição da missão
O programa simula um Dojo da Academia Backend onde aprendizes podem treinar, descansar, ganhar experiência e consultar seu nível. A classe principal apenas executa o fluxo, enquanto a classe `AprendizTecnica` controla seu próprio comportamento.

## Como executar
1. Abrir o projeto no IntelliJ.
2. Executar a classe `DojoTecnicas`.
3. Verificar a saída no console.

## Resultado esperado
O console deve mostrar aprendizes sendo criados, treinando, descansando, ganhando experiência e exibindo status inicial e final.
